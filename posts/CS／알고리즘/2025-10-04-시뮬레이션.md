## 4.  dx, dy 테크닉
어떤 격자에서 동, 서, 남, 북으로 이동할 때

```
dx = {1, 0, -1, 0};
dy = {0, -1, 0, 1};

x += dx[dir_num];
y += dy[dir_num];
```
로 단순화할 수 있다.

특히 이동 방향이 많아지면 더 유리
(x 이동방향부터 먼저 만들기)

- 방향 회전을 요구하는 경우 
	- 회전 방향에 맞게 dx, dy를 설계
- 동서남북 -> 순서에 맞게 설계

```c++
dx = {0, 1, 0, -1};
dy = {1, 0, -1, 0};

dir = (dir + 1) % 4     // 우회전
dir = (dir + 4 - 1) % 4 // 좌회전
dir = (dir + 2)         // 뒤돌기
```
